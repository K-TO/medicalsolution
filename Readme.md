# Medical Solution

Medical solution es un mini proyecto con arquitectura de N capas, en las cuales se reparten las diferentes tareas que requiere un sistema con conexión a base de datos y servicios rest.


# Archivos

Los archivos con especificaciones del proyecto se encuentran en .\Utilidades y Entregables.

## Requisitos

* .Net framework 2.2
* MSSQL Server
* Angular ^6.0.0

## Switch to another file

All your files and folders are presented as a tree in the file explorer. You can switch from one to another by clicking a file in the tree.

# Instalación

* Para la instalación solo es necesario compilar la solución, esta a su vez, se encargara de instalar las dependencias y nugget requeridos para su funcionamiento.
* En el caso de la base de datos, el tomara el servidor **localhost** (esta configuración podra ser cambiada en ./MedicalSolution.Presentation/appsettings.json).
